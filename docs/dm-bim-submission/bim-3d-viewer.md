### Instructions

1. **Navigate**: Click and drag to rotate the view
2. **Zoom**: Use mouse wheel to zoom in/out
3. **Pan**: Right-click and drag to pan
4. **Reset Camera**: Click the "Reset Camera" button to return to default view
5. **Toggle Grid**: Show/hide the ground grid and axes

### Setup

Make sure the file `sample.ifc` is placed in the `assets/models/` directory.
